Conclusions

This section offers conclusions for the project as a whole.

This project was a perfect introduction for me to both the worlds of Deep Learning as well as Computer Vision, and despite it being relatively simple in terms of implementations, its performance is enough to be used in a production environment if the project was to be further developed.

Also, this project showed me the implications of deep learning in the world of computer vision, which are endless considering the current state of AI generated content and media (especially image and video), and I will keep on exploring this field as it is fascinating and it is going nowhere.
